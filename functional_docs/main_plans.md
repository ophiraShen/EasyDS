# EasyDS - 基于多智能体的智能教学系统

## 1. 系统概述

EasyDS 是一个基于大语言模型的多智能体协同教学系统，专注于考研数据结构教育。系统采用费曼学习法的教学理念，通过路由、学生和教师三层智能体模型，突破传统"问答式"AI辅导教学范式，构建"讲解-反馈-修正-强化"的教学闭环。

### 1.1 核心特点

- 基于费曼学习法的引导式教学
- 三层智能体模型（路由、学生、教师）
- 基于章节的结构化问题数据库
- GRPO优化的教师智能体
- 自适应学习路径生成

### 2.1 智能体组成

1. **路由智能体**
   - 评估用户讲解回答的正确性与完整性
   - 基于章节和知识点关联分析
   - 决策选择学生智能体或教师智能体
   - 动态调整教学路径

2. **学生智能体**
   - 针对用户讲解提出深度追问
   - 模拟学习者困惑点
   - 引导用户进行更清晰的阐述
   - 基于相关知识点扩展问题

3. **教师智能体**
   - 实现费曼学习法的引导式教学
   - 纠错与方向性指导
   - 总结反馈与知识点强化
   - 通过GRPO优化的计算推理能力

### 2.2 核心模块

1. **多智能体协同框架**
   - 基于LangGraph/LangChain的智能体通信
   - 教学流程状态管理
   - 智能体角色定义与转换

2. **结构化问题数据库**
   - 基于章节的问题组织
   - 知识点间关联关系
   - 扩展问题链接
   - 自适应学习路径生成

3. **教师智能体优化系统**
   - 基于DeepSeek-R1-Distill-Qwen-14B
   - GRPO奖励机制设计
   - 双指标评估体系：答案正确性+步骤合理性
   - 计算推理能力增强

## 3. 实现计划

### 3.1 第一阶段：多智能体系统搭建

1. **基础框架搭建**
   - LangGraph/LangChain环境配置
   - 智能体通信协议设计
   - 会话流程实现

2. **智能体开发**
   - 路由智能体实现
     * 路由决策Prompt设计
     * 问题数据库接口集成
     * 状态转换逻辑实现
   - 学生智能体实现
     * 追问策略设计
     * 上下文理解能力
     * 相关知识点扩展能力
   - 教师智能体实现
     * 纠错引导机制
     * 知识点讲解模板
   - 智能体协同调试

3. **教学流程优化**
   - 状态流转逻辑完善
   - 边界情况处理
   - 用户体验优化

### 3.2 第二阶段：结构化问题数据库构建

1. **数据库设计**
   - 章节结构设计
   - 问题模型定义（包含知识点、关联问题）
   - 关系类型定义（前置知识、相关知识、扩展知识）

2. **问题数据准备**
   - 基于教材章节划分问题
   - 知识点提取与关联
   - 问题难度分级
   - 参考答案编写

3. **数据库实现**
   - 数据存储方案选择（SQL/NoSQL）
   - 数据导入与验证
   - 查询接口实现
   - 与智能体系统集成

### 3.3 第三阶段：教师智能体GRPO优化

1. **数据准备**
   - 计算类题目收集
     * 近10年考研真题精选
     * 数据增强扩展
   - 数据标注与处理
     * 双指标评估标准建立
     * 训练集划分

2. **GRPO训练框架搭建**
   - 基座模型准备（DeepSeek-R1-Distill-Qwen-14B）

     * 答案正确性评估（7分）
     * 步骤合理性评估（3分）
   - 训练流程实现

3. **模型训练与评估**
   - 参数优化
   - 性能评估
   - 模型集成

### 3.4 第四阶段：系统集成与测试

1. **模块整合**
   - 多智能体系统部署
   - 问题数据库接入
   - 优化后教师智能体集成

2. **系统测试**
   - 功能测试
   - 性能测试

3. **系统优化**

## 4. 技术栈

- LangGraph/LangChain: 智能体框架
- PyTorch: 模型训练框架
- SQLite/MongoDB: 问题数据库存储
- FastAPI: 后端服务
- Vue.js: 前端界面
- DeepSeek-R1-Distill-Qwen-14B: 基座模型
- PyMuPDF: PDF文档处理
- Spacy/HanLP: 自然语言处理
